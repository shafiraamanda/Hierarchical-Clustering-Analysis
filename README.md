# Hierarchical Clustering Analysis

## Overview

This project uses hierarchical clustering to group data based on similarity. The goal is to identify natural clusters and explore patterns within the dataset.

## Objectives

* Segment data into meaningful groups
* Analyze similarity between data points
* Identify patterns using clustering

## Method

The analysis applies **Agglomerative Hierarchical Clustering**, where data points are gradually merged into clusters based on distance similarity.

A dendrogram is used to visualize the clustering process and determine the optimal number of clusters.

## Tools

Python, Pandas, Scikit-learn, Scipy, Matplotlib

## Output

The result is a set of clusters that group similar data points and reveal underlying patterns in the data.

## How to Run

Open the notebook file and run all cells.
Make sure the required libraries are installed (pandas, numpy, scikit-learn, scipy, matplotlib).

## Conclusion

The clustering results provide meaningful groupings and help identify patterns within the dataset.
