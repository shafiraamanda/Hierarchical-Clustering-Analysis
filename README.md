# Hierarchical Clustering Analysis

## Overview

This project applies hierarchical clustering to group data based on similarity. The objective is to identify natural clusters and explore patterns within the dataset.

## Objectives

* Segment data into meaningful groups
* Analyze similarity between data points
* Identify patterns using clustering techniques

## Methodology

The project uses **Agglomerative Hierarchical Clustering**, where data points are progressively merged into clusters based on distance similarity.

Key components:

* Distance metric (e.g., Euclidean distance)
* Linkage method (e.g., Ward linkage)
* Dendrogram for cluster visualization

## Analysis Steps

* Data preprocessing (cleaning and normalization)
* Distance matrix calculation
* Hierarchical clustering implementation
* Dendrogram visualization
* Determination of optimal number of clusters
* Cluster interpretation

## Tools

Python, Pandas, Scikit-learn, Scipy, Matplotlib

## Output

The analysis produces clusters that group similar data points, helping to reveal underlying structures and patterns in the data.

## How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
2. Open and run the notebook file.

## Conclusion

Hierarchical clustering successfully groups the data into meaningful clusters, providing insights into similarities and potential segmentation within the dataset.
